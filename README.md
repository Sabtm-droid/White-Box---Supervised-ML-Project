# White-Box---Supervised-ML-Project
The project demonstrates Supervised ML models.
1) White Box - Supervised ML Project - DSBPT1 - Classification.ipynb: this file for classification models (Logistic and KNN). The purpose of this model to classify and predict the score of the customers based on certain features.
2) White Box - Supervised ML Project - DSBPT1 - Regression.ipynb: this file for regression models (linear and KNN). The purpose of the model is to predict which region has more weight on the global sales
